# Overview #

The GetGames.php and GetGamesList.php API calls can accept an optional "platform" parameter when using the "name" parameter.

**An example of this is:**

_http://thegamesdb.net/api/GetGamesList.php?name=x-men&platform=Nintendo+Entertainment+System+(NES)_

You can use the following API call to see a listing of all available Platforms: http://thegamesdb.net/api/GetPlatformsList.php
